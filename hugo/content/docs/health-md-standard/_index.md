Open YAML-based standard for structuring healthcare information optimized for Large Language Models. Privacy-first, clinically-accurate, MCP-compatible format.

**Perfect for:** Healthcare AI, EHR systems, telemedicine, personal health records.

## Why EIR Format?

- **LLM-Optimized** ‚Äî YAML structure perfect for AI parsing
- **MCP Compatible** ‚Äî Load into Claude, ChatGPT via Model Context Protocol
- **Privacy-First** ‚Äî Built-in anonymization patterns
- **Clinical-Accurate** ‚Äî Preserves medical context, uses ICD-10, SNOMED, LOINC

## Getting Started

1. Read the [Specification](https://github.com/BirgerMoell/health-md-standard)
2. Install the parser: `pip install health-md`
3. Use the [Validator](/eir-open/projects/health-md-standard/validator.html) to validate EIR files
